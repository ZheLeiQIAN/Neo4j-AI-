## 附：文档使用建议

### A.1 按学习阶段的文档使用顺序

| 学习阶段 | 推荐文档 | 目标 |
|----------|---------|------|
| 入门 | 入门指南 → Cypher Manual → GraphAcademy Cypher 课程 | 掌握 Neo4j 基本操作与 Cypher 查询 |
| 进阶 | GDS Manual → APOC 文档 → GDS 基础课程 | 掌握图算法与数据工程 |
| 高级 | GenAI 插件 → Vector Index → GraphRAG 指南 | 掌握图原生 AI 与 GraphRAG |
| 工程化 | Operations Manual → 驱动文档 → GitHub 仓库 | 掌握生产部署与应用开发 |

### A.2 版本兼容性提示

- GDS 库版本与 Neo4j 数据库版本需匹配，参见 [GDS 兼容性矩阵](https://neo4j.com/docs/graph-data-science/current/installation/#_compatibility)；
- APOC 分为 `apoc-core`（内置）与 `apoc-extended`（需单独安装），安装时注意区分；
- GenAI 插件从 Neo4j 5.18 起作为官方插件提供，早期版本需使用 Labs 版本。
