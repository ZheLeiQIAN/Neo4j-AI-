# 中英对照术语表

> 本术语表按英文字母顺序排列，涵盖知识库全部核心术语。每个术语包含英文原文、中文翻译与简要定义，便于在阅读英文文献时对照检索。
> **使用说明**：术语首字母相同的条目按字母顺序排列；以缩写或专有名词（如 APOC、Cypher、node2vec）开头的术语归入对应字母段。

## A

| 英文术语 | 中文翻译 | 简要定义 |
|----------|---------|---------|
| A* Algorithm | A*算法 | 结合启发式估计的最短路径算法，通过实际代价与估计代价之和引导搜索方向 |
| Aggregator Function | 聚合函数 | GraphSAGE 中将邻居特征聚合为单一向量的函数，包括均值、LSTM、最大池化等 |
| All Pairs Shortest Path | 全对最短路径 | 计算图中所有节点对之间最短路径的算法 |
| APOC (Awesome Procedures On Cypher) | APOC | Neo4j 社区维护的实用过程与函数库，提供数百个 Cypher 扩展功能 |

## B

| 英文术语 | 中文翻译 | 简要定义 |
|----------|---------|---------|
| Betweenness Centrality | 介数中心性 | 衡量节点出现在其他节点对最短路径上的频率，反映节点作为"桥梁"的程度 |
| Bipartite Graph | 二分图 | 节点可分为两个不相交的子集，且每条边连接分属不同子集的节点的图 |
| Bolt Protocol | Bolt 协议 | Neo4j 设计的二进制网络协议，用于客户端与数据库之间的高效通信 |
| Bounding Box | 边界框 | 用矩形框标注图像中物体位置与范围的坐标表示 |

## C

| 英文术语 | 中文翻译 | 简要定义 |
|----------|---------|---------|
| Centrality | 中心性 | 衡量节点在图中重要性的指标族，包括度中心性、接近中心性、介数中心性、PageRank 等 |
| Closeness Centrality | 接近中心性 | 衡量节点到图中所有其他节点的平均最短距离的倒数，反映节点的全局可达性 |
| Connected Component | 连通分量 | 图中最大的连通子图，分量内任意两节点可达，分量间不可达 |
| Connected Graph | 连通图 | 任意两个节点之间都存在路径的无向图 |
| Constraint | 约束 | 保证数据完整性的规则，如唯一性约束确保某属性值不重复 |
| Cosine Similarity | 余弦相似度 | 两个向量夹角的余弦值，衡量向量方向的相似性，忽略向量长度 |
| Cycle | 环 | 起点与终点为同一节点的路径，即闭合路径 |
| Cypher | Cypher | Neo4j 的声明式图查询语言，采用模式匹配语法表达图查询 |

## D

| 英文术语 | 中文翻译 | 简要定义 |
|----------|---------|---------|
| DeepWalk | DeepWalk | 最早的基于随机游走的节点嵌入方法，将随机游走序列类比为句子，用 Word2vec 训练嵌入 |
| Degree | 度 | 与某节点直接相连的边的数量；有向图中分为入度与出度 |
| Degree Centrality | 度中心性 | 以节点的度（连接数）直接作为重要性度量，是最简单的中心性指标 |
| Dijkstra's Algorithm | Dijkstra 算法 | 在非负权重图中寻找单源最短路径的经典算法 |
| Directed Graph | 有向图 | 边具有方向性的图，每条边从源节点指向目标节点 |

## E

| 英文术语 | 中文翻译 | 简要定义 |
|----------|---------|---------|
| Edge | 边 | 图中连接两个节点的元素，表示节点间的关系，也称为链接（Link） |

## F

| 英文术语 | 中文翻译 | 简要定义 |
|----------|---------|---------|
| FastRP (Fast Random Projection) | FastRP | 基于随机投影的高效图嵌入算法，无需迭代，时间复杂度近似线性，适用于大规模图 |

## G

| 英文术语 | 中文翻译 | 简要定义 |
|----------|---------|---------|
| Graph | 图 | 由节点（顶点）集合与边（关系）集合构成的数学结构，用于表示实体间的拓扑关系 |
| Graph Attention Network (GAT) | 图注意力网络 | 引入注意力机制的图神经网络，自适应地为不同邻居分配不同权重 |
| Graph Catalog | 图目录 | GDS 中管理图投影的组件，负责图的创建、删除、导出与统计 |
| Graph Convolutional Network (GCN) | 图卷积网络 | Kipf & Welling (2017) 提出的图神经网络，通过一阶谱近似的逐层传播规则实现高效的图卷积 |
| Graph Data Science (GDS) | 图数据科学 | 运用图算法与图机器学习从图结构数据中提取知识与洞察的学科领域，也是 Neo4j 算法库的名称 |
| Graph Embedding | 图嵌入 | 将图结构数据（节点、边、子图）映射到低维向量空间的技术，是连接图与机器学习的桥梁 |
| Graph Laplacian | 图拉普拉斯 | 图的拉普拉斯矩阵（度矩阵减邻接矩阵），其特征值与特征向量揭示图的谱性质 |
| Graph Neural Network (GNN) | 图神经网络 | 直接在图结构数据上运行的神经网络，通过邻居信息聚合学习节点或图的表示 |
| Graph Projection | 图投影 | GDS 将数据库中的图数据加载到内存中专用图结构的过程，是运行算法的前置步骤 |
| Graph Sampling | 图采样 | 从大图中抽取子集用于训练的技术，是 GNN 可扩展性的关键，如邻居采样、层采样 |
| Graph-Native AI | 图原生 AI | 以图结构为核心数据表示的 AI 范式，将知识图谱与关系推理能力深度融入 AI 模型 |
| GraphRAG (Graph RAG) | 图检索增强生成 | 结合知识图谱与大语言模型的检索增强生成方法，通过图结构实现结构化检索与推理 |
| GraphSAGE | GraphSAGE | 归纳式图神经网络，通过学习可复用的邻居聚合函数为未见节点生成嵌入 |

## H

| 英文术语 | 中文翻译 | 简要定义 |

| Hybrid Retrieval | 混合检索 | 结合多种检索策略（如向量语义检索与图结构遍历）的检索方法，GraphRAG 的核心特征 |

| Image Retrieval | 图像检索 | 根据查询条件（文本、图像、场景图等）从图像数据库中查找相关图像的任务 |

### A.2 翻译约定

1. **专有名词与算法名**：如 Cypher、node2vec、GraphSAGE、PageRank、Louvain、DeepWalk、FastRP 等保留英文原名，不做翻译；
2. **缩写词**：如 GNN、GDS、GCN、GAT、RAG、LLM、APOC 等在首次出现时给出全称与中文翻译，后续使用缩写；
3. **一词多义**：如 Node 与 Vertex 同义，均译为"节点/顶点"；Connected Component 既指图论概念也指 GDS 算法，定义中统一说明；
4. **学术术语**：遵循图论与机器学习领域的通行译法，参考全国科学技术名词审定委员会标准。
