## 三、GNN 专项课程

> 图神经网络与图深度学习方向的顶级课程资源，适合具备深度学习基础的学习者。

### 3.1 Stanford CS224W: Machine Learning with Graphs

| 项目 | 详情 |
|------|------|
| 课程主页 | http://web.stanford.edu/class/cs224w/ |
| 视频资源 | https://www.youtube.com/playlist?list=PLoROMvodv4rPLKxipqhjhPg7TYK52hoR9 |
| 课程资料 | http://snap.stanford.edu/class/cs224w-2021/ |
| 开设学期 | 每年秋季（Stanford 校内），视频全年公开 |
| 费用 | 视频/课件免费，无证书 |
| 关联章节 | 第4篇全篇（`04-01` 至 `04-08`） |

**课程简介**：由 Jure Leskovec 教授（node2vec、GraphSAGE 作者）主讲，是图机器学习领域最权威的课程。涵盖图结构表示、节点嵌入（DeepWalk、node2vec）、GNN 架构（GCN、GraphSAGE、GAT）、图生成、知识图谱嵌入等全栈内容。课程作业基于 PyTorch Geometric，实操性强。

**学习建议**：
- 预计投入 60-80 小时（含作业）；
- 建议配合论文精读：node2vec（KDD 2016）、GCN（ICLR 2017）、GraphSAGE（NeurIPS 2017）、GAT（ICLR 2018）；
- 作业涉及 Colab 编程，需 Python 与 PyTorch 基础。

## 五、学习路径与课程组合建议

### 5.1 按目标的课程组合

| 学习目标 | 推荐课程组合 | 预计周期 | 对应知识库篇目 |
|----------|------------|---------|--------------|
| 掌握 Neo4j 基础开发 | GraphAcademy: Neo4j Fundamentals → Cypher Fundamentals → Data Modeling → Python 开发 | 3-4 周 | 第1篇、第6篇 |
| 掌握 GDS 图分析 | GraphAcademy: GDS Fundamentals → GDS with Python → Coursera: Graph Analytics | 4-6 周 | 第2篇、第4篇 |
| 深入 GNN 研究 | Stanford CS224W → Graph Representation Learning（McGill）→ PyG 实践 | 10-12 周 | 第4篇全篇 |
| 构建 GraphRAG 应用 | GraphAcademy: Vector Search & RAG → Neo4j GenAI → LangChain 集成 | 3-4 周 | 第3篇全篇 |
| LIS 领域图技术应用 | Neo4j Fundamentals → GDS Fundamentals → 知网论文研读 → 数字人文实践 | 6-8 周 | 第1-3篇 + 第5篇 |

### 5.2 认证路径建议

```
入门阶段（0-4周）
  └─ GraphAcademy: Neo4j Fundamentals + Cypher Fundamentals
       └─ 获得课程证书

进阶阶段（5-8周）
  └─ GraphAcademy: GDS Fundamentals + Data Modeling
       └─ 报考 Neo4j Certified Professional 认证

高级阶段（9-16周）
  └─ Stanford CS224W + GraphAcademy: GDS with Python
       └─ 报考 Neo4j Graph Data Science Certification

应用阶段（17-20周）
  └─ GraphAcademy: Vector Search & RAG + GenAI
       └─ 构建个人 GraphRAG 项目
```

### 5.3 时间投入参考

| 学习强度 | 每周投入 | 完成基础课程 | 完成认证 | 完成 CS224W |
|----------|---------|------------|---------|------------|

## 附：课程选择快速参考

### A.1 我是 LIS 研究生，从未接触过图数据库，该怎么开始？

1. **第1步**（2周）：GraphAcademy - Neo4j Fundamentals + Cypher Fundamentals（免费）
2. **第2步**（2周）：GraphAcademy - Data Modeling + Importing CSV Data（免费）
3. **第3步**（2周）：GraphAcademy - GDS Fundamentals（免费）
4. **第4步**（持续）：结合本知识库第1-3篇阅读中文论文

### A.2 我有 Python 与机器学习基础，想学 GNN，该怎么学？

1. **第1步**（4周）：Stanford CS224W 视频 + 作业
2. **第2步**（2周）：PyTorch Geometric 官方教程
3. **第3步**（2周）：GraphAcademy - GDS with Python
4. **第4步**（持续）：精读 node2vec、GCN、GraphSAGE、GAT 四篇论文

### A.3 我想构建一个 GraphRAG 应用，需要学什么？

1. **第1步**（1周）：GraphAcademy - Vector Search & RAG with Neo4j
2. **第2步**（1周）：GraphAcademy - Neo4j GenAI with Python
3. **第3步**（1周）：LangChain × Neo4j 集成教程
4. **第4步**（2周）：阅读 Edge et al. (2024) GraphRAG 论文 + 微软 GraphRAG 仓库实践
